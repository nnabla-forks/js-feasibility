# Modulo Krinkle Tiling Viewer

arXiv:2506.07638 で提案された "Modulo Krinkle Tilings" をブラウザ上で生成・可視化するためのツールです。
Pythonによる生成ロジックをJavaScriptに移植し、インタラクティブなパラメータ操作とリアルタイムな描画を実現しています。

## 特徴

- **リアルタイム生成**: パラメータ ($m, k, t$) を変更すると即座にタイリングが再計算されます。
- **3つの表示モード**:
  - **Prototile (Single)**: 基本となる1枚のタイル（プロトタイル）とその境界数列を表示します。
  - **Wedge (Layout)**: プロトタイルを三角形に配置した "Wedge" を表示します。
  - **Tiling (Full)**: Wedgeを回転・配置して円形の領域全体を埋めるタイリングを表示します。
- **インタラクティブ操作**: マウスドラッグによる移動、ホイールによるズームが可能です。
- **詳細な可視化**:
  - 辺番号、タイル番号、Wedge番号の表示切り替え。
  - ホバー時のWedgeハイライトおよび深度（Row）ハイライト。
  - Offset Mode（$n = 2(tk - m)$）のサポート。

## 使い方

本アプリケーションはローカル環境で動作するように設計されています（ビルド不要）。

1. このディレクトリ内の `index.html` をブラウザ（Chrome, Firefox, Safari等）で開いてください。
2. 画面左上のコントロールパネルからパラメータを操作します。

## パラメータ解説

| パラメータ | 説明 | 数式的な意味 |
| --- | --- | --- |
| **m (Step)** | 数列のステップ幅（勾配） | $y \equiv m \cdot x \pmod k$ |
| **k (Mod)** | モジュラス（法） | 周期やサイズの基準となります |
| **t (Period)** | 周期係数 | 対称性 $n$ を決定します ($n = k \times t$) |
| **Offset** | オフセットモード | 有効時、計算式が変化します ($n = 2(tk - m)$) |
| **Display Mode** | 表示モード切り替え | Prototile / Wedge / Tiling |
| **Rows (Depth)** | 生成する深さ | Wedgeの段数やタイリングの密度に影響します |

## 操作方法

- **視点移動**: 画面内をドラッグ
- **ズーム**: マウスホイール（スクロール）
- **表示切替**:
  - `Show Edge #`: 辺のインデックスを表示 (Prototileモード)
  - `Show Wedge #`: Wedgeのインデックスを表示 (Tilingモード)
  - `Show Tile #`: 個々のタイルのインデックスを表示

## ファイル構成

- `index.html`: エントリーポイント。UI構造が記述されています。
- `app.js`: アプリケーションの全ロジック（UI制御、描画、計算）。
- `style.css`: スタイルシート。ダークモード/グラスモーフィズムデザイン。
