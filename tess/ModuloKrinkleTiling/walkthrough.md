# Modulo Krinkle Tiling エディタ ウォークスルー

Modulo Krinkle Tiling エディタの実装が完了しました。このツールを使用すると、論文 "Modulo Krinkle Tiling" (arXiv:2506.07638) に基づく非周期的なタイリングを視覚化できます。

## 機能

### 1. インタラクティブ・キャンバス
- **レンダリング**: HTML5 Canvasを使用した高速な描画。
- **ナビゲーション**:
    - **パン**: ドラッグして移動。
    - **ズーム**: ホイールで拡大縮小。

### 2. コントロールパネル
- 各種パラメータを調整してタイリングをリアルタイムに生成します。
- **パラメータの意味**:
    - `モジュロ (M)`: 論文パラメータ `k` に相当。基本的な周期やプロトタイルの複雑さを決定します。
    - `パラメータ A`: 論文パラメータ `m` に相当。ステップサイズを決定します。
    - `パラメータ B`: 論文パラメータ `n` に相当。全体の回転対称性の分割数を決定します。
    - `パラメータ C`: 現在は未使用（将来の拡張用）。

### 3. タイリングロジック (`krinkle.js`)
- **アルゴリズム**:
    1. **プロトタイル生成**: `l_j = (j * m) % k` の数列に基づいて、閉じたウェッジ形状（ポリゴン）を生成します。
    2. **フロント追跡**: 既に配置されたタイルの境界（フロント）を管理し、次に配置するタイルの位置と回転を計算します。
    3. **ウェッジ配置**: `n` 個のウェッジを計算された位置に配置し、平面を埋め尽くします。

## 実行方法

1. Webブラウザで `tess/ModuloKrinkleTiling/index.html` ランチャーを開きます。
   ```bash
   open tess/ModuloKrinkleTiling/index.html
   ```
2. パラメータを調整し、「タイリング更新」を押してパターン生成を試してください。
   - 推奨例: M=7, A=3, B=14 (論文の図2の例)
