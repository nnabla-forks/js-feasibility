# Modulo Krinkle Tiling エディタ ウォークスルー

Modulo Krinkle Tiling エディタの初期バージョンを実装しました。このツールを使用すると、整数パラメータに基づいて非周期的なタイリングを視覚化できます。

## 機能

### 1. インタラクティブ・キャンバス
- **レンダリング**: 高性能な2D描画のためにHTML5 Canvasを使用しています。
- **ナビゲーション**:
    - **パン**: クリックしてドラッグすると、キャンバス内を移動できます。
    - **ズーム**: マウスホイールを使用して拡大縮小できます。

### 2. コントロールパネル
- **グラスモーフィズムデザイン**: ぼかし効果を備えたモダンな半透明パネル。
- **パラメータ**:
    - `Modulo (M)`: セクターの数/対称性を制御します。
    - `パラメータ A, B, C`: 形状生成に影響を与える整数（現在はプレースホルダーのパターンに接続されています）。

### 3. タイリングロジック (`krinkle.js`)
- 現在は、パイプラインを実証するために放射状の「ファン」パターンを生成する**プレースホルダーアルゴリズム**が含まれています。
- 数式が完全に書き起こされたら、論文の正確な「remainder sequence（余剰数列）」アルゴリズムと簡単に置き換えられるように設計されています。

## 実行方法

1. Webブラウザで `tess/ModuloKrinkleTiling/index.html` ファイルを開きます。
   ```bash
   open tess/ModuloKrinkleTiling/index.html
   ```
2. コントロールパネルでパラメータを調整し、「更新」ボタンをクリックします。

## ファイル構成

- `index.html`: メインのエントリポイント。
- `style.css`: スタイル。
- `src/main.js`: アプリケーションコントローラー。
- `src/renderer.js`: Canvasインタラクション。
- `src/krinkle.js`: タイリング生成ロジック。
